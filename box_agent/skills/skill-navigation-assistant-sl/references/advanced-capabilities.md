# 增强能力说明

## 1. 技能健康评分

在基础 normal/warning/error 之外，为每个技能计算 0-100 健康分。

扣分项包括：
- 缺少 `SKILL.md`
- YAML front matter 缺失或不可解析
- 缺少 description 或中文用途说明
- 缺少版本号
- 无法分类
- 缺少唤起方式

输出应包含：健康分、问题原因、修复建议。

## 2. 最近使用 / 高频推荐

如果用户提供 usage JSON，则基于使用记录输出：
- 最近使用技能
- 高频技能
- 建议置顶技能

如果没有使用记录，则降级为基于健康分和通用性推荐。

## 3. 技能组合方案

当任务具有多阶段意图时，输出：
- 阶段名称
- 推荐技能
- 唤起方式
- 组合理由
- 建议执行顺序

## 4. 技能缺口分析

统计本地技能类别覆盖，输出：
- 当前强项类别
- 缺口类别
- 健康风险技能
- SkillHub 搜索关键词

## 5. 我的技能说明书

基于扫描结果生成 Markdown 说明书，包括：
- 技能总览
- 分类清单
- 置顶建议
- 当前任务工作流
- 技能库强项与缺口
- 健康风险与修复建议
- 下一步建议
