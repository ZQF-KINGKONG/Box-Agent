# 技能健康检查

## 检查项

- 是否存在 `SKILL.md`。
- YAML front matter 是否存在。
- YAML front matter 是否可解析。
- name、description、version 字段是否存在。
- description 是否足够说明技能用途。
- 是否可以推断唤起方式。
- 技能包结构是否明显异常。

## 健康状态

- 正常：关键文件和注册元数据完整。
- 警告：可以识别，但存在描述不足、中文名缺失或字段缺失。
- 异常：缺少 SKILL.md、YAML 不可解析、缺少关键注册字段。

## 输出建议

健康检查只给出维护建议，不应在用户未确认的情况下修改、覆盖或删除任何技能文件。
