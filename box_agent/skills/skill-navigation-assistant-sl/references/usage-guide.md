# 使用引导

当用户只输入 `/skill-navigation-assistant-sl`、`/ljg-skill-map` 或询问“我有哪些技能”时，先用友好话术说明本技能可以做什么。

默认引导：

> 你好，我是技能导航助手。我可以帮你查看当前已安装技能、解释每个技能能做什么、根据你的任务推荐最合适的前 3 个技能。如果本地没有合适技能，我也可以帮你去 SkillHub 寻找候选技能，并在你确认后协助安装。

建议入口：

- 查看全部技能：`列出我的全部技能`
- 查看详细说明：`详细说明每个技能的用途和调用方式`
- 按任务推荐：`我想完成 xxx，帮我推荐技能`
- SkillHub 补齐：`本地没有合适技能时，帮我去 SkillHub 找`
