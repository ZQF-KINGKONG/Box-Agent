# 偏好处理

## 识别偏好

当用户表达以下内容时，视为世界杯内容偏好：

- “我喜欢内马尔”
- “我关注巴西”
- “以后世界杯多写巴西”
- “重点看姆巴佩/梅西/贝林厄姆”
- “我更关心南美球队”
- “简报里多加关键人员信息”

## 应用偏好

当已知用户偏好时：

1. 在简报或预告中增加相关球队、球员、教练、地区的内容权重。
2. 适合时加入“用户关注重点”或“关注对象动态”小节。
3. 对用户关注对象必须尽量补充关键人员信息，包括状态、角色、伤停、战术位置、近期表现。
4. 不能因为用户偏好而遗漏重大赛事信息。
5. 不能为了迎合偏好而夸大球队实力、隐瞒不利信息或扭曲判断。

## 更新偏好

如果用户明确表示稳定偏好，例如“我喜欢内马尔，巴西”“以后都多关注巴西”，应：

1. 简短确认：“已记住，后续世界杯内容会优先关注巴西和内马尔。”
2. 如果宿主环境支持长期记忆，并且符合记忆规则，可保存为用户偏好。
3. 后续生成内容时默认体现该偏好。

## 示例

用户：“我喜欢内马尔，巴西。”

后续世界杯简报应：

- 优先补充巴西队动态。
- 如相关且可核实，补充内马尔状态、角色、伤病/恢复、与球队战术的关系。
- 仍然客观覆盖其他重大比赛和关键人物。
