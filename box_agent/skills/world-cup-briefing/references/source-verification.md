# 来源与不确定性处理

## 来源优先级

1. FIFA 官方网站、官方赛程页、官方比赛中心。
2. 参赛队、赛事组织方、官方社媒或新闻稿。
3. 权威通讯社、主流体育数据源和可靠媒体报道。
4. 中文媒体、球迷社区和社交媒体只能作为辅助线索，不可单独作为确认事实。

## 必须核验的信息

- 比分、赛程、开球时间、比赛阶段和场地。
- 进球、助攻、红黄牌、点球、乌龙、换人等关键事件。
- 伤停、停赛、复出、首发、阵容、晋级/出线形势。
- 用户关注球员或球队的最新状态。

## 不确定性表达

- 未找到官方或权威确认：写“未检到官方确认”。
- 来源冲突：写“来源存在冲突，待确认”，并避免下结论。
- 预测、判断、球迷视角：必须和已确认事实分开。
- 不得为了页面完整度编造比分、人员、伤病、采访或战术安排。

## HTML 中的呈现

HTML 简报底部必须包含“信息来源与待确认”区域，说明更新时间、来源类型，以及哪些内容仍需刷新确认。
