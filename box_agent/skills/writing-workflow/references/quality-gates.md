# 质量门禁

## 发布前检查清单

1. 主题明确：文章是否能用一句话说明要表达什么？
2. 观点鲜明：是否有判断，而不是资料堆砌？
3. 结构顺畅：段落是否有递进，读者是否会迷路？
4. 素材充分：是否有案例、事实、经验或推理支撑？
5. 表达自然：是否存在模板腔、AI 腔、空话套话？
6. 风格统一：是否符合用户样本或指定语气？
7. 事实可靠：具体人名、机构、年份、数据、因果是否可核查？
8. 平台适配：是否适合公众号、朋友圈、小红书、视频号或演讲场景？
9. 风险可控：是否有声誉、隐私、版权、事实风险？
10. 交付完整：是否包含用户要求的标题、正文、摘要、发布建议等？

## 风险降级表达

- 未核实事实：改为“据用户提供材料”“若情况属实”“有待进一步核实”。
- 负面评价：改为“可能反映出管理机制问题”“需要更多证据支持”。
- 数据不确定：删除具体数字或标注“需来源确认”。
- 私人原文：概括、脱敏、抽象，不直接复制。
